# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it through [GitHub's private vulnerability reporting](https://github.com/steph-dm/MyUta/security/advisories/new).

Include a description, steps to reproduce, and potential impact. I'll acknowledge within 48 hours and aim to fix confirmed issues within 7 days.

## Scope

- Authentication and session management
- Data access controls (cross-user data leaks)
- GraphQL injection or abuse
- API key exposure

Not in scope: self-hosted instances with misconfigured environments, social engineering.
